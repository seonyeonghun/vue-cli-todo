# todo-app

## Project setup
```
1. node.js LTS install
2. vue install -g @vue/cli
3. vue create todo-app
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
